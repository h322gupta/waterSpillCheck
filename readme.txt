********************************************************************************

This model is intended to predict images with water accumulation

********************************************************************************
************************How to Run**********************************************

In order to check the model predictions, run test_predict.py file
This file takes two arguments
	1. Image folder path (path is provided in the file as default)
	2. model path        (path is provided in the file as default)
	
********************************************************************************
***********************Model output*********************************************

A csv file will be generated which contains image name along with the class
